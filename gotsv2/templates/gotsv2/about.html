About the Game of Thrones Web Scraping Demo App:

How It Works

    A list of all character names is scraped from Wikipedia using the BeautifulSoup packge for Python 3: List of A Song of Ice and Fire Characters
    My script saves the list of names to the database and uses them to generate a list of URLs for individual character pages, eg. en.wikipedia.org/wiki/Ned_Stark
    If a character doesn't have their own character page they are filtered out at this stage, because it's not certain what information is available about them. However, if they were to get their own character page later on, the new page would be picked up when the gotscrape database updates.
    Further web scraping of the individual character pages returns information about the characters' genders and titles.
    Then I used Django and Charts.js to display this data graphically.
    Much more information about each character could be scraped in the same way from the 'infobox' class div on the character pages, as Wikipedia uses standardised formatting, but I just wanted to focus on a couple of types of information in order to demonstrate web scraping.
